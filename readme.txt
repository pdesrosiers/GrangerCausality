DESCRIPTION:

	Granger is a small MATLAB package that infers significant causal links between time series.   

REQUIREMENTS:

	MATLAB 2016 or higher.

INSTALLATION:

	Please first run setup.m, then run example.m.

MAIN FUNCTIONS:

	inferNetworkWithGranger.m
	granger.m
